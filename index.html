Guide to the [UNIX/Linux server] Sys Admin Body of Knowledge

Intended audience

  • Anyone who wants to be a System Administrator,
  • Junior System Administrators who want to make sure their foundation is solid for advancement to more senior levels,
  • Intermediate/Advanced/Senior System Administrators advancing to more senior levels,
  • Trainers and Educators

Scope

This is a guide to the professional literature of the field of System Administration.

This document does not include practice exercises. However, practical experience is essential to success in the field. The reader is encouraged to practice what they read about. A future document (or series) will include guided practical exercises.

This guide includes an overview of the normative literature in the field - publications that lay out the norms of the field, or what is generally agreed upon (i.e. best practices). The purpose of that is to encourage normative literature, which essential for professionalization.

How to use this guide

The materials should be studied and practiced in sequence.

The Core contains essential reading. The Normative list contains additional best practices. Start with the Core and then the Normative list.

Supplement and expand your knowledge using the Supplemental list.

Advance your knowledge further using the Advanced list.
