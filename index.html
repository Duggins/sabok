Guide to the Sys Admin Body of Knowledge (SABOK)

Intended audience

  • Anyone who wants to be a UNIX/Linux System Administrator,
  • Junior System Administrators who want to make sure their foundation is solid for advancement to more senior levels,
  • Intermediate/Advanced/Senior System Administrators advancing to more senior levels.

Scope

This is a guide to the professional literature of the field of
System Administration.

It starts with links to definitions of what is system administration,
what are the boundaries of the field, and what other fields are adjacent
to system administration.

This document does not include practice exercises.  However, practical
experience is essential to success in the field.  The reader is encouraged
to practice what they read about.  A future document (or series) will
include guided practical exercises.

This guide includes an overview of the normative literature in the field -
publications that lay out the norms of the field, or what is generally
agreed upon (i.e. best practices).  The purpose of that is to encourage
normative literature, which essential for professionalization.

How to use this guide

The materials should be studied and practiced in sequence. The Core contains essential reading. The Normative list contains additional best practices. Start with the Core and then the Normative list. Supplement and expand your knowledge using the Supplemental list. Advance your knowledge further using the Advanced list.

Core Reading List

    Smith, Roderick W. Linux Essentials Indianapolis, IN: Wiley, 2012. Print.

    An introduction to Linux and Open Source.
    Elsner, Tobias et al. Linux Essentials: The LPI Introductory ProgrammeDarmstadt, Germany: Linup Front, 2012. http://shop.linupfront.com/

    An alternative introduction to Linux and Open Source. Courseware for LPI Linux Essentials program. Free for non-commercial use.
    Shotts, William. The Linux Command Line: A Complete Introduction First Edition. San Francisco: No Starch, 2012. Print.
    Shotts, William. The Linux Command Line: A Complete Introduction Second Edition, 2013. http://linuxcommand.org

    A complete introduction to the Linux Command Line. We have started to recommend this book instead of the dated "The UNIX Programming Environment". Available in PDF and dead tree versions.
    Nemeth, Evi et al. Unix and Linux System Administration Handbook, 4th ed. Upper Saddle River, NJ: Prentice Hall, 2012. Print.

    "This is a comprehensive guide to the care and feeding of UNIX and Linux systems. The authors present the facts along with seasoned advice and real-world examples."
    -- Pat Parseghian
    Frisch, Æleen. Essential System Administration, 3rd ed. Beijing: O'Reilly, 2002. Print.
    Limoncelli, Thomas A., Christine Hogan, and Strata R. Chalup. The Practice of System and Network Administration, 2nd ed. Harlow: Addison-Wesley, 2007. Print.

    A comprehensive overview of the practice of system administration.
